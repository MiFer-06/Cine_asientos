simula una sala de cine de verdad
la idea es que cualquiera pueda ver qué asientos están libres y cuáles no
Primero le pides al programa cuántas filas y columnas tendrá la sala, el código crea como una matriz para representar los asientos
Si un asiento está libre aparece con una L y si está ocupado se ve una X 
Hay un menú con opciones para que puedas ver la sala completa, reservar un asiento, liberarlo o ver las estadísticas de cuántos asientos libres y ocupados hay en total
